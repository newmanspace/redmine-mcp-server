## 九、SQL 查询示例

### 查询项目成员
```sql
SELECT 
    u.login,
    STRING_AGG(r.name, ', ') as roles
FROM warehouse.ods_project_memberships pm
JOIN warehouse.ods_users u ON pm.user_id = u.user_id
JOIN warehouse.ods_project_member_roles mr ON pm.membership_id = mr.membership_id
JOIN warehouse.ods_roles r ON mr.role_id = r.role_id
WHERE pm.project_id = 341
GROUP BY u.user_id, u.login;
```

### 查询 Issue 分布
```sql
SELECT 
    s.name as status,
    COUNT(*) as count
FROM warehouse.ods_issues i
JOIN warehouse.ods_issue_statuses s ON i.status_id = s.status_id
WHERE i.project_id = 341
GROUP BY s.status_id, s.name;
```
