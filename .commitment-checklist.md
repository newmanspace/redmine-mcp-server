# Developer Commitment Checklist

## Before Git Commit
- [ ] Has the user explicitly confirmed to commit?
- [ ] Have I reviewed all changes?
- [ ] Is the commit message clear and concise?

## Before Code Generation
- [ ] Have I confirmed the requirements with the user?
- [ ] Is the naming convention correct?
- [ ] Are all comments in English?

## Before Execution
- [ ] Did I explain what I'm going to do?
- [ ] Did I wait for user confirmation?

## Code Quality
- [ ] No Chinese characters in SQL/Python code
- [ ] Naming follows project conventions
- [ ] Files are properly organized
